# MenuButton

Included are styles that wrap the [ReachUI MenuButton CSS Selectors](https://reacttraining.com/reach-ui/menu-button#menubutton-css-selectors) and provide a custome property interface for overrides.

## Basic

Menu Button can be styled using the `data-select` selector for identital look and feel to OS selects.

<Preview>
  <Story name="Basic">
    <Menu>
      <MenuButton data-select>Basic, non-ruled</MenuButton>
      <MenuList>
        <MenuItem onSelect={() => alert("called function")}>
          Calls function
        </MenuItem>
        <MenuLink to="#one">Navigates to URL</MenuLink>
        <MenuLink to="#two">Navigates to URL</MenuLink>
      </MenuList>
    </Menu>
  </Story>
</Preview>

## Basic with Group Rules

`MenuItems` can be subdivided into groups using `data-reach-menu-rule`.

<Preview>
  <Story name="Group Rules">
    <Menu>
      <MenuButton data-select>Basic, group ruled</MenuButton>
      <MenuList>
        <MenuItem onSelect={() => alert("called function")}>
          Calls function
        </MenuItem>
        <hr data-reach-menu-rule />
        <MenuLink to="#one">Navigates to URL</MenuLink>
        <MenuLink to="#two">Navigates to URL</MenuLink>
        <div data-reach-menu-rule />
        <MenuLink to="#two">Navigates to URL</MenuLink>
      </MenuList>
    </Menu>
  </Story>
</Preview>

## Selected Menu Item

`selected` MenuItems background color can be set with the custom property `--reach-menu-item--border-color_selected`.

<Preview>
  <Story name="Selected Menu item">
    <Menu>
      <MenuButton data-select>Basic, group ruled</MenuButton>
      <MenuList style={{ "--reach-menu-item--background-color_selected": "red" }}>
        <MenuLink to="#one">Navigates to URL</MenuLink>
        <MenuLink to="#two">Navigates to URL</MenuLink>
      </MenuList>
    </Menu>
  </Story>
</Preview>
